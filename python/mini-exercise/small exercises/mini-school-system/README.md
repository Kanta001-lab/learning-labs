## Mini School System

## Dependencies

- Python 3.8+
- No external packages


## Concepts Learned

- object.attribute – direct attribute access.
- object`index` – if the attribute is a list/tuple.
- object`key` – if the attribute is a dictionary.
- You can chain any number of these: obj.attr1`index`.attr2`key`.attr3.
- You can assign through chains (if the final target is assignable).
- You can use them in conditions and loops.

## Breif Note: Chained Attribute Access

Chained attribute access(.) that connects one thing to another. In Python,
everything is an object, and objects have attributes(data) and methods
(functions). when you see something.another, you're accessing an attribute
or method of something.

## Example:

Think of it as exploring a house:

- You walk into a house.
- Inside, there's a kitchen.
- Inside the kitchen, there's an oven.
- Inside the oven, there's a pizza.

*Each dot takes you a level deeper.*
