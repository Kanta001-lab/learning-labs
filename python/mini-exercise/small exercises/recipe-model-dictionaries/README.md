# Recipe Model (Dictionaries)

A small Python exercise that demonstrates how dictionaries can be used as lookup tables by mapping pizza types and toppings to emoji representations.

### Dependencies:
- Python 3.8+
- No external libraries

### Usage:
    python "03 Recipe Model (Dictionaries).py"


This Recipe Model from 02, but now uses dictionaries as "lookup tables" to map pizza types and toppings to their emoji representations.

## Concepts Practiced

- Dictionary creation
- Key-value lookups
- Fallback values with `.get()`
- Data mapping

## Why dictionaries are useful:
- Fast lookups (no need to search through lists)
- Clear relationships between keys and values
- Easy to add new items without changing logic
- Perfect for menus, settings, or configuration data

## Real-world Analogy:
A dictionary is like a real dictionary:
- You look up a word (key)
- You find its definition (value)
- If the word isn't there, you have a fallback value can be returned.
