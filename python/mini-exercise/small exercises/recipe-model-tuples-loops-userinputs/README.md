## Recipe Model (Tuples, Loops and UserInput)

A small python exercise that demostrates Dictionaries, Tuples, UserInput and Loops.


## Dependencies:
- Python 3.8+
- No external libraries

## Usage:
    python "05 Recipe Model (Tuples, Loops and UserInput).py"

## Concepts Learned:
  1. Dictionaries with Tuple values
  2. For loops for displaying menus
  3. While loops for input validation
  4. User input handling
  5. Function definition and calling
  6. Price calculation with size multipliers
