## 02 Recipe Model (Functions).py

## Understanding Functions - Using the Recipe Analogy

### Dependencies:
- Python 3.8+
- No external libraries

### Usage:
    python "02 Recipe Model (Functions).py"

### Concepts Learned:
A function is like a recipe:
 The function demonstrates how functions work using a donut-making analogy. Each call creates a new donut with different toppings.

1. NAME: What you call it (e.g., "make_donut")
2. INGREDIENTS: Parameters you pass in (e.g., glaze_type)
3. INSTRUCTIONS: The code inside the function
4. RESULT: What the function returns
