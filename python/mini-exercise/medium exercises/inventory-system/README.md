## Inverntory System

A complete Inventory system that add and update stocks and tracks
transactions, notifies on low threshold alerts with generated
sales reports and statistics.



## Key Features:

- Object-oriented design with Inventory class
- Uses defaultdict and Counter from collections module
- Times stamps every sale automatically with stock updates
- Private restock method
- Generated sales report

## Dependencies:
- Python 3.8+
- collections (standard library - no install needed)
- datetime (standard library - no install needed)
